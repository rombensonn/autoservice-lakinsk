# Лендинг «Автомойка на Карла Маркса»

Демо-лендинг для автомойки, шиномонтажа, автосервиса и вклейки стекол по адресу: Лакинск, ул. Карла Маркса, 24.

## Структура

- `index.html` - одностраничный лендинг.
- `css/styles.css` - адаптивные стили.
- `css/redesign.css` - визуальный слой Liquid Glass.
- `js/main.js` - меню, подстановка услуги в форму, AJAX-отправка.
- `images/` - изображения лендинга.
- `robots.txt`, `sitemap.xml` - базовая SEO-заготовка.

## Запуск

```bash
php -S localhost:8000
```

Откройте `http://localhost:8000`.

## Публикация

Статическая версия подготовлена для GitHub Pages. Форма отображается в интерфейсе, но серверная отправка заявок на GitHub Pages не выполняется.
